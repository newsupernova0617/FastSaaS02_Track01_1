엑셀과 앱은 둘 다 가계부를 쓰는 도구지만, 사용 경험은 꽤 다릅니다. 무엇이 더 좋다기보다 **어떤 사람이 오래 쓸 수 있느냐**가 더 중요합니다.

<QuickSummary title="빠른 판단 기준">
  <SummaryItem label="엑셀 추천">
    직접 표를 설계하고 통제하는 감각이 중요하다면 엑셀이 잘 맞습니다.
  </SummaryItem>
  <SummaryItem label="앱 추천">
    기록 속도, 이동 중 입력, 중간 점검, 차트 확인이 중요하다면 앱이 유리합니다.
  </SummaryItem>
</QuickSummary>

## 한눈에 보는 차이

| 기준 | 엑셀 가계부 | 앱 가계부 |
| --- | --- | --- |
| 입력 속도 | 느릴 수 있음 | 빠름 |
| 모바일 사용성 | 약한 편 | 강함 |
| 커스터마이징 | 매우 강함 | 서비스 범위 내에서 가능 |
| 차트/요약 | 직접 만들어야 함 | 기본 제공되는 경우가 많음 |
| 지속성 | 관리 습관이 강한 사람에게 유리 | 초보자와 바쁜 사람에게 유리 |

## 엑셀 가계부의 장점

- 원하는 표와 수식을 직접 설계할 수 있음
- 항목 구조를 세밀하게 통제할 수 있음
- 월별 비교, 합계, 비율 계산을 세세하게 조정할 수 있음

엑셀은 "내 기준대로 만들고 싶다"는 사람에게 강합니다. 이미 숫자 관리에 익숙하고, 기록을 주기적으로 정리하는 습관이 있는 사람이라면 만족도가 높을 수 있습니다.

## 엑셀 가계부의 약점

- 기록 진입 장벽이 상대적으로 높음
- 모바일에서 바로 입력하기 불편할 수 있음
- 차트나 요약을 직접 만들어야 하는 경우가 많음
- 며칠만 밀려도 업데이트가 무거워짐

특히 가계부를 처음 시작하는 사람에게는 "자유도"가 장점보다 피로로 느껴질 수 있습니다.

## 앱 가계부의 장점

- 빠른 입력
- 이동 중 사용이 편함
- 차트, 리포트, 카테고리 요약을 바로 보기 쉬움
- 기록이 끊겼을 때 다시 복귀하기가 비교적 쉬움

앱은 속도와 지속성에서 강합니다. 기록 습관이 아직 약하거나, 중간중간 소비 흐름을 자주 다시 보고 싶은 사람에게 유리합니다.

## 앱 가계부의 약점

- 서비스마다 구조와 철학이 다름
- 커스터마이징이 엑셀보다 제한적일 수 있음
- 자동 연동, 분석 기능, 보안 수준이 앱마다 크게 다를 수 있음

<Callout title="주의" tone="warning">
  앱은 모두 비슷해 보이지만 실제 차이는 입력 경험, 자동 분류 정확도, 보안 정책에서 크게 납니다. 설치 전에 비교가 필요합니다.
</Callout>

## 어떤 사람에게 엑셀이 맞을까

아래에 가까우면 엑셀이 맞습니다.

- 직접 표를 설계하는 게 귀찮지 않다
- PC에서 정리하는 시간이 있다
- 월말 정산형 루틴에 익숙하다
- 분석보다 통제감이 더 중요하다

## 어떤 사람에게 앱이 맞을까

아래에 가까우면 앱이 맞습니다.

- 기록을 자주 빼먹는다
- 이동 중에도 바로 적고 싶다
- 차트나 요약을 빠르게 보고 싶다
- 월말이 아니라 중간중간 확인하고 싶다

## 결국은 지속성이 더 중요합니다

엑셀이든 앱이든 핵심은 오래 쓰는 것입니다. 기능이 아무리 좋아도 입력이 무거우면 멈추고, 구조가 아무리 예뻐도 다시 안 열어보면 의미가 없습니다.

그래서 선택 기준은 단순합니다.

1. 내가 지금 바로 기록할 수 있는가
2. 일주일 뒤에도 같은 방식으로 다시 열 수 있는가
3. 월말이 아니어도 흐름을 확인할 수 있는가
