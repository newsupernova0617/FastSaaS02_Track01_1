## 가계부 앱을 고를 때 먼저 봐야 할 5가지

1. 입력 속도
2. 자동 분류 정확도
3. 리포트와 차트
4. 동기화와 백업
5. 개인정보와 신뢰성

이 다섯 가지가 맞아야 오래 갑니다. 기능이 많아도 입력이 느리면 금방 닫게 되고, 자동 연동이 좋아도 보안이 불안하면 계속 쓰기 어렵습니다.

## 비교할 만한 5가지 선택지

여기서는 특정 서비스 순위보다, 사용자가 실제로 많이 고민하는 앱 유형을 기준으로 비교합니다.

| 유형 | 잘 맞는 사람 | 장점 | 약점 |
| --- | --- | --- | --- |
| 자동 연동형 앱 | 기록을 자주 빼먹는 사람 | 누락이 적고 흐름 보기 쉬움 | 연결 과정과 보안 검토가 필요 |
| 심플 기록형 앱 | 처음 시작하는 사람 | 입력이 가볍고 부담이 적음 | 분석 기능이 약할 수 있음 |
| 자산관리형 앱 | 자산 전체를 같이 보고 싶은 사람 | 자산과 지출을 함께 보기 좋음 | 화면이 무거울 수 있음 |
| 엑셀 대체형 앱 | 표 중심 정리를 좋아하는 사람 | 구조화와 필터링이 편함 | 초보자에게는 무거울 수 있음 |
| 대화형 AI 가계부 | 분류 선택 자체가 귀찮은 사람 | 자연어 입력, 빠른 조회 | 익숙해지는 데 시간이 필요할 수 있음 |

### 1. 자동 연동형 앱

- 입력이 빠름
- 누락이 적음
- 전체 소비 흐름을 보기 쉬움

대신 연결 과정이 번거로울 수 있고, 보안과 개인정보를 더 신경 써야 합니다.

### 2. 심플 기록형 앱

- 처음 쓰기 쉬움
- 필요한 기능만 있어 덜 피곤함
- 모바일에서 바로 적기 편함

대신 분석 기능이 약하거나, 사용자별 커스터마이징이 적을 수 있습니다.

### 3. 자산관리형 앱

- 돈 흐름을 넓게 볼 수 있음
- 자산과 지출을 함께 관리 가능

대신 처음 진입할 때 화면이 복잡할 수 있고, 가계부만 가볍게 쓰고 싶은 사람에게는 과할 수 있습니다.

### 4. 엑셀 대체형 앱

- 정리 감각이 좋음
- 표 중심으로 보기 쉬움
- 분석과 필터링이 편할 수 있음

대신 초보자에게는 입력이 무거울 수 있습니다.

### 5. 대화형 AI 가계부

- 자연어 입력이 쉬움
- 분류를 직접 고르는 부담이 적음
- 소비 패턴을 질문으로 바로 확인 가능

대신 아직은 새롭고, 어떤 기준으로 분류되는지 익숙해져야 합니다.

## 실제로는 이렇게 고르면 됩니다

- 기록을 자주 빼먹는다 -> 자동 연동형 또는 AI 가계부
- 분류 기준이 자꾸 헷갈린다 -> 심플 기록형 또는 AI 가계부
- 자산까지 같이 보고 싶다 -> 자산관리형
- 표와 통제를 중시한다 -> 엑셀 대체형
