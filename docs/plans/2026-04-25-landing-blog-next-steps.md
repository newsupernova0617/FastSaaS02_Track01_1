# 랜딩/블로그 남은 작업 정리

작성일: 2026-04-25

## 현재 상태

- Astro 랜딩 `/`에 블로그 진입 버튼과 최신 글 미리보기 섹션이 들어가 있음
- `/blog` 목록 페이지와 개별 상세 페이지가 동작함
- `robots.txt`, `sitemap.xml`, 기본 OG/Twitter 메타, JSON-LD가 적용됨
- 현재 작성된 글
  - `가계부 쓰는 법 완벽 가이드`
  - `생활비 예산 세우는 법`
  - `가계부 항목 분류 꿀팁`
  - `엑셀 가계부 vs 앱 가계부 비교`
  - `사회초년생 첫 월급 관리 A to Z`

## 우선순위 높은 남은 작업

### 1. 핵심 블로그 글 추가 작성

다음 글을 우선 작성한다.

- `가계부 3일 만에 포기하는 이유와 해결법`
- `가계부 앱 추천 TOP 5 솔직 비교`
- `가계부 앱 개인정보 안전할까`
- `자취생 생활비 30만원으로 버티기`
- `소비 패턴 파악하는 5가지 방법`

목표:

- 검색형 + 전환형 글 숫자를 최소 8~10개까지 확장
- 홈 최신 글 섹션과 `/blog` 목록이 더 풍부하게 보이도록 만들기

### 2. 내부링크 구조 강화

현재 글들 사이 연결은 시작됐지만 아직 허브 구조가 완성되진 않음.

정리할 링크 흐름:

- `가계부 쓰는 법` -> `생활비 예산 세우는 법`
- `가계부 쓰는 법` -> `가계부 항목 분류 꿀팁`
- `가계부 쓰는 법` -> `엑셀 가계부 vs 앱 가계부 비교`
- `생활비 예산 세우는 법` -> `가계부 항목 분류 꿀팁`
- `생활비 예산 세우는 법` -> `사회초년생 첫 월급 관리`
- `가계부 3일 만에 포기하는 이유` -> `가계부 쓰는 법`
- `가계부 앱 추천` -> `엑셀 vs 앱 비교`
- `개인정보 안전할까` -> `앱 추천`

목표:

- 정보형 글 -> 비교형 글 -> 전환형 글 흐름 만들기
- 검색 유입 후 앱 관련 글로 자연스럽게 이동시키기

### 3. 블로그 CTA 구조 추가

현재는 글 본문은 있지만 전환용 CTA는 아직 약함.

추가할 것:

- 본문 하단 공통 CTA 박스
- `출시 알림 받기` 또는 `앱 보기` 링크
- 관련 글 2~3개 묶음

권장 문구 방향:

- `기록을 더 쉽게 이어가고 싶다면`
- `예산과 지출 흐름을 한 번에 보고 싶다면`
- `앱형 가계부 방식도 같이 비교해보세요`

### 4. 블로그 목록 탐색 개선

현재 `/blog`는 단순 목록 구조임.

추가 후보:

- 카테고리 필터
- 태그별 빠른 진입
- 추천 글 / 최신 글 분리
- 인기 글 섹션

우선순위:

- 1차: 카테고리 필터
- 2차: 추천 글 / 최신 글 구분

### 5. OG 이미지 체계화

현재는 기본 OG 이미지를 중심으로 사용 중임.

필요 작업:

- 블로그 공통 OG 템플릿 제작
- 포스트별 제목 반영 방식 정의
- 카테고리별 색상 또는 배지 규칙 정의

목표:

- 검색 결과 공유 클릭률 개선
- 카카오톡 / X / Discord 공유 시 비주얼 일관성 확보

### 6. 분석 도구 연결

현재는 어떤 글이 유입되고 전환되는지 추적할 기반이 없음.

필수 검토:

- `GA4` 또는 `Plausible`
- 블로그 글 조회수
- `/blog` -> `/` -> `waitlist` 이동 흐름
- CTA 클릭 이벤트

최소 측정 포인트:

- `blog_post_view`
- `blog_cta_click`
- `home_blog_card_click`
- `waitlist_submit`

### 7. 홈 `/` 추가 개선

지금 홈에는 최신 글 미리보기 3개가 들어가 있음.

다음 개선 후보:

- 카테고리 바로가기 추가
- `가계부 가이드`, `앱 비교`, `초년생 돈관리` 같은 주제별 진입 카드
- 최신 글 외에 `추천 글` 한 개를 더 강조

## 실행 순서 제안

1. `가계부 3일 만에 포기하는 이유와 해결법` 작성
2. `가계부 앱 추천 TOP 5 솔직 비교` 작성
3. `가계부 앱 개인정보 안전할까` 작성
4. 블로그 하단 CTA 공통 컴포넌트 추가
5. `/blog` 카테고리 필터 추가
6. 분석 도구 연결
7. OG 이미지 체계화

## 완료 기준

- 블로그 글 8개 이상 발행
- 핵심 글 사이 내부링크 구조 완성
- 각 글 하단에 CTA 존재
- `/blog` 탐색성이 현재보다 개선됨
- 분석 도구로 블로그 유입과 CTA 클릭 추적 가능
