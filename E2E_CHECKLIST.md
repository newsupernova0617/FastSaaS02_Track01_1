## 1️⃣ Chat 기본 기능 (Chat Features)

### 1.1 세션 관리 (Session Management)

- [ ] **세션 생성**
  - [ v] 새로운 세션 생성 (POST /api/sessions)
  - [ x] 첫 메시지로 자동 제목 생성 확인
  - [v ] 세션 ID가 정상적으로 반환되는지 확인
  - [ v] 여러 세션을 동시에 생성 가능한지 확인

- [ ] **세션 목록 조회**
  - [ v] 모든 세션 조회 (GET /api/sessions)
  - [v ] 세션 목록이 생성 순서 역순(DESC)으로 정렬되는지 확인
  - [v ] 세션 메타데이터(id, title, createdAt, updatedAt) 확인

- [ ] **세션 상세 조회**
  - [v ] 특정 세션 조회 (GET /api/sessions/:id)




- [ ] **세션 삭제**
  - [v ] 세션 삭제 (DELETE /api/sessions/:id)
  - [ v] 관련 메시지 cascade 삭제 확인
  - [ v] 삭제된 세션 재조회 불가 확인



### 1.3 Plain Text 응답 (Non-Financial Queries)

- [ ] **Plain Text 감지 및 응답**
  - [v ] 금융 관련 없는 쿼리 입력: "안녕하세요", "오늘 날씨어때", "파이썬 배우는 법" 등
  - [v ] AI 응답 actionType이 'plain_text'인지 확인
  - [v ] 응답이 PLAIN_TEXT_FALLBACK 메시지임을 확인
  - [v ] 트랜잭션이 생성되지 않았는지 확인

- [ ] **금융 쿼리와 Plain Text 구분**
  - [v ] "5000원 커피 사줬어" → actionType='create'
  - [ ] "지난달 식비는?" → actionType='read'
  - [ ] "리포트 만들어줘" → actionType='report'
  - [ v] "안녕, 뭐해?" → actionType='plain_text'

## 6️⃣ 통합 플로우 (Integration Flows)

### 6.1 완전한 Chat 세션 플로우

```
1. 새 세션 생성 → "April 1st Financial Planning"
2. 첫 메시지: "4월 예산을 계획해줘"
3. AI 응답: "현재 데이터 기반으로..."
4. 거래 생성: "월급 300만원"
5. 거래 조회: "이번달 지출은?"
6. 리포트 생성: "4월 분석해줘"

8. Stats 탭에서 저장된 리포트 확인
9. Plain Text: "파이썬은 뭐야?" → plain_text 응답
10. 세션 종료
```
